584-400-0120 Teacher Candidate Performance Assessments

(1) Preservice Candidates: All Oregon preservice teacher candidates must complete a Commission-approved teacher performance assessment, as required in the Program Review and Standards Handbook, if the Commission has approved a performance assessment for the endorsement area.

(2) If the EPP requires a teacher performance assessment, the EPP has the option to use one of the following: (a) A State-adopted Teacher Performance Assessment, as described in the Program Review and Standards Handbook; (b) A Commission-approved Local Assessment Option, as provided in subsection (5); or (c) Options as defined in the Multiple Measures Guidance Information.

(3) Two or More Endorsements (Preservice Candidates): If a preservice candidate is completing two or more separate endorsement programs, the preservice candidate is required to complete a teacher performance assessment for only one of the endorsements, as provided: (a) If one of the endorsements requires a Commission-adopted performance assessment and the other does not, the candidate must complete the Commission-adopted performance assessment. (b) If all of the endorsement areas require a Commission-adopted performance assessment, the EPP, in consultation with the candidate, must select the most appropriate teacher candidate performance assessment area to complete the performance assessment requirement. (c) Notwithstanding subsection (b), if one of the endorsement subjects is Elementary Multiple Subjects (EMS), then the candidate must complete the performance assessment in EMS. (d) Notwithstanding subsection (b) and (c), if the endorsement subjects include EMS and SPED: Gen, then the EPP, in consultation with the candidate, must select the most appropriate performance assessment in either EMS or SPED: Gen.

(4) Adding Endorsements (Licensed Teachers): Licensed teachers adding endorsements to existing licenses are not required to complete a teacher performance assessment.

(5) Commission-approved Local Assessment Option: To qualify as a local assessment option, a teacher performance assessment must align to the Commission-approved Local Assessment Option Framework.
