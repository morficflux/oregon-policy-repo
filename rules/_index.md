# Administrative Rules (OAR) — index

Oregon Administrative Rules: regulations with the force of law, adopted under ORS 183,
published in OARD by the Secretary of State. Structure: chapter (3 digits) → division
(3 digits) → rule (4 digits). Directory layout mirrors this: `rules/<chapter>/<division>/`.

## How to find the right rule

- **Chapter 125** is DAS. **Division 800** is State Information Security (e.g.,
  OAR 125-800-0020, the planned first ingest — the rule tier behind DAS policy
  107-004-052).
- Rule history is encoded in Administrative Order Numbers (AONs), e.g.
  "DAS 8-2006, f. & cert. ef. 12-28-06".
- Upstream changes arrive via the monthly Oregon Bulletin (first business day).
