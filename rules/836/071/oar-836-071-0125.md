## Full text

836-071-0125 Period for Completion of License Application An application for a license is invalid if the license for which the application was made has not been granted on or before the last day of the ninth month after the date on which the application was filed with the Director.

Statutory/Other Authority: ORS 731.244 Statutes/Other Implemented: ORS 744.001
