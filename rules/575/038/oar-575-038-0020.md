## Full text

575-038-0020 Program Elements Governed by Need Grant Rules The following common elements of the Opportunity Grant Program and the State Grant Supplemental Award Program are governed by existing administrative rules for the Opportunity Grant Program:

(1) Definitions, see Oregon Administrative Rules (OAR) 575-030-0005.

(2) Residency, see OAR 575-031-0005.

(3) Financial need, see OAR 575-031-0010.

(4) Eligible program, see OAR 575-031-0016.

(5) Enrollment, see OAR 575-031-0020.

(6) Conditions of award, see OAR 575-031-0030.

(7) Renewal, see OAR 575-031-0075.

(8) Award priority, see OAR 575-031-0040.

(9) Disbursement, see OAR 575-031-0045.

(10) Record keeping, see OAR 575-031-0050.

(11) Appeals, see OAR 575-031-0055.

Statutory/Other Authority: ORS 348 Statutes/Other Implemented: HB 5030(6) & 1995
