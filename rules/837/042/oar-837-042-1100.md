## Full text

837-042-1100 Definitions

(1) "Board" means the Oregon Fire Code Advisory Board.

(2) “Interest Form” means an application for consideration submitted by a person expressing interest in serving on an Oregon Code Advisory Board or subcommittee.

(3) “Oregon Fire Code Advisory Board” means a board consisting of appointed members who make recommendations on Oregon Fire Code issues to the State Fire Marshal and provide advice and guidance on issues of code enforcement, regional disputes, policies, and emerging topics that affect the fire protection and life safety of Oregonians.
