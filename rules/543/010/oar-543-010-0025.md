## Full text

543-010-0025 Loan Periods The State Library Board authorizes the State Librarian to set loan periods for State Library materials. Loan periods will be available to the public in a State Library policy.

Statutory/Other Authority: ORS 357.026(1) Statutes/Other Implemented: ORS 357.026(1)
