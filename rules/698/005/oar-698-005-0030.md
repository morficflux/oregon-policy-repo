# Application Requirements (OAR 698-005-0030)

## Full text

698-005-0030 Application Requirements Applications must be submitted on the most current format required by OWEB. An explanation must accompany the application if any of the information required on the application cannot be provided.

Statutory/Other Authority: ORS 541.984, ORS 541.988 & ORS 541.989 Statutes/Other Implemented: ORS 541.977-541.989
