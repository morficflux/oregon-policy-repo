918-460-0015 Amendments to the Oregon Structural Specialty Code

(1) The Oregon Structural Specialty Code is amended pursuant to OAR chapter 918, division 8. Amendments adopted during the code cycle for inclusion into the Oregon Structural Specialty Code are placed in this rule, showing the section reference, a descriptive caption, and a short description of the amendment.

(2) Effective April 1, 2026, Sections 3404.5.2.1, 3404.5.2.1.2, 3404.5.2.4, 3405.3.3, 3407.6.2, and 3408.5.2 of Chapter 34 of the Oregon Structural Specialty Code are amended to provide an exception from having to comply with new construction provisions for certain repairs and alterations to existing buildings in select locations of the state where new ground snow load provisions apply. [Publications: Publications referenced are available from the agency.]

Statutory/Other Authority: ORS 447.231, ORS 455.030, ORS 455.110, ORS 455.496 & ORS 455.477 Statutes/Other Implemented: ORS 455.110 & ORS 455.477
