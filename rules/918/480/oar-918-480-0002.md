# Specialty Code Name Change (OAR 918-480-0002)

## At a glance

## Full text

918-480-0002 Specialty Code Name Change All references in OAR chapter 918 to the Oregon One- and Two-Family Dwelling Specialty Code mean the Oregon Residential Specialty Code. [Publications: Publications referenced are available for review at the agency.]

Statutory/Other Authority: ORS 455.610 Statutes/Other Implemented: ORS 455.610
