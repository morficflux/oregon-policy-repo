## Full text

918-440-0012 Amendments to the Oregon Mechanical Code The Oregon Mechanical Specialty Code is amended pursuant to OAR chapter 918, division 8. Amendments adopted for inclusion into the Oregon Mechanical Specialty Code are placed in this rule, showing the section reference, a descriptive caption, and a short description of the amendment. [Publications: Publications referenced are available from the agency.]

Statutory/Other Authority: ORS 455.030 Statutes/Other Implemented: ORS 455.110
