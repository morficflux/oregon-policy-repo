## Full text

918-690-0440 Plumbing Section The chief plumbing inspector is charged with the organization and enforcement of the Oregon Plumbing Specialty Code created under ORS 447.020 and defined as a Specialty Code under ORS 455.010, as a part of the State Building Code under ORS 455.010. [Publications: Publications referenced are available from the agency.]
