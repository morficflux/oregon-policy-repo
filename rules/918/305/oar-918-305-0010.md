## Full text

918-305-0010 Scope of the Electrical Specialty Code The Electrical Specialty Code applies to all nonexempt electrical installations except as covered by the electrical provisions of the Oregon Residential Specialty Code and is inspected by an Electrical Specialty Code inspector. [Publications: Publications referenced are available from the agency.]

Statutory/Other Authority: ORS 479.730 Statutes/Other Implemented: ORS 479.730
