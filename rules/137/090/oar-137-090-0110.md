# Information (OAR 137-090-0110)

## Full text

137-090-0110 Information Contributions To the extent possible, all criminal intelligence maintained in CIU files must display the names and phone numbers of persons and agencies providing the information. When anonymity is requested by a contributor, a contributor code number may be used. All contributor code numbers will be provided and retained by the CIUS. When a contributor’s name identification is difficult to obtain, it will suffice to describe the contributor in general terms. All information obtained from the public domain will be identified by document name, date and page number. In addition to identifying the source, the manner in which the source obtained the information is described.
