## Full text

436-001-0610 Duty to Forward Misdirected Request If an employer or insurer receives a written request for hearing or administrative review from a worker, and the request should have been filed with the division, the employer or insurer must promptly forward the request to the division.

Statutory/Other Authority: ORS 656.726(4) Statutes/Other Implemented: ORS 656.704
