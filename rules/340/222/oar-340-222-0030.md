# Definitions (OAR 340-222-0030)

## At a glance

## Full text

340-222-0030 Definitions The definitions in OAR 340-200-0020, 340-204-0010 and this rule apply to this division. If the same term is defined in this rule and 340-200-0020 or 340-204-0010, the definition in this rule applies to this division. NOTE: This rule is included in the State of Oregon Clean Air Act Implementation Plan that EQC adopted under OAR 340-200-0040.

Statutory/Other Authority: ORS 468.020 & 468A Statutes/Other Implemented: ORS 468A
