## Full text

635-080-0044 Columbia Basin Unit The Columbia Basin Unit, Number 44, is that area beginning at Condon, north on State Highway 19 to Oregon-Washington state line at Arlington; northeast along state line to State Highway 11; southwest on State Highway 11 to Pendleton; southwest on U.S. Highway 395 to State Highway 74; southwest on State Highway 74 to Heppner; southwest on State Highway 207 to State Highway 206; west on State Highway 206 to Condon, point of beginning.

Statutory/Other Authority: ORS 496 Statutes/Other Implemented: ORS 496
