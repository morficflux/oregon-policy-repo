# Planning (PL) — Statewide IT Control Standards family

## At a glance

**[SUMMARY]** The Planning (PL) control family of Oregon's Statewide Information
Technology Control Standards (January 2024). It defines 6 base controls (plus
enhancements, 7 entries total) that agencies must meet under DAS Policy
107-004-052 and ORS 276A.300. Control numbering and structure follow NIST SP 800-53
Rev. 5. For binding control text, consult the source PDF — this file is an inventory only.

## Control inventory

**[SUMMARY]** Control identifiers and titles as printed in the PL section of the
source document:

| Control | Title |
|---|---|
| PL-1 | Policy and Procedures |
| PL-2 | Security Plans |
| PL-4 | Rules of Behavior |
| PL-4(1) | Rules of Behavior | Social Media and External Site/application Usage Restriction |
| PL-8 | Security Architecture |
| PL-10 | Baseline Selection |
| PL-11 | Baseline Tailoring |
