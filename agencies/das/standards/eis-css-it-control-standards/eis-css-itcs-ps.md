# Personnel Security (PS) — Statewide IT Control Standards family

## At a glance

**[SUMMARY]** The Personnel Security (PS) control family of Oregon's Statewide Information
Technology Control Standards (January 2024). It defines 9 base controls (plus
enhancements, 10 entries total) that agencies must meet under DAS Policy
107-004-052 and ORS 276A.300. Control numbering and structure follow NIST SP 800-53
Rev. 5. For binding control text, consult the source PDF — this file is an inventory only.

## Control inventory

**[SUMMARY]** Control identifiers and titles as printed in the PS section of the
source document:

| Control | Title |
|---|---|
| PS-1 | Policy and Procedures |
| PS-2 | Position Risk Designation |
| PS-3 | Personnel Screening |
| PS-3(3) | Personnel Screening | Information Requiring Special Protective Measures |
