# Statewide Information Security Plan (8/1/2018)

## At a glance

**[SUMMARY]** The statewide plan document required by ORS 276A.300(2) and listed as an
exhibit of DAS Policy 107-004-052; agencies "must comply with the Statewide Information
Security Plan, statewide policies and security standards" (per that policy). The plan
covers the information security program and risk management, security policies,
organization of information security (roles, governance), and related program areas.
Dated 8/1/2018; contact is the Enterprise Security Office (CSS's predecessor name). Note
OAR 128-030-0010's broader definition: the "State Information Security Plan" is a
compilation including statutes, rules, policies, and plans.

## Structure (from the source table of contents)

**[SUMMARY]** Sections include: Introduction; Document Scope; Purpose and Applicability
(adoption, target audience, authority, leadership and commitment); Terms and Definitions
(must/should/will usage); Information Security Program and Risk (risk assessment; risk
management framework); Information Security Policies (management direction;
rules/policies/standards/procedures hierarchy; policy review); Organization of
Information Security (roles and responsibilities, governance). (Source: contents pages.)
