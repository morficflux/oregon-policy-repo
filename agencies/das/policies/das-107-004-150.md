# Cloud and Hosted Systems (DAS Statewide Policy 107-004-150)

## At a glance

**[SUMMARY]** Requires agencies to analyze benefits, costs, and risks before contracting
for cloud or hosted services, assess provider readiness, and plan protection of state
information and financial assets. Effective 5/1/2019 (supersedes the 2016 version);
approved by Terrence Woods, State CIO. Note the interaction with 107-004-052's
offshoring prohibition: offshore cloud/hosting is not allowed without a State CIO
exception.

## Key provisions

### Purpose

> **[VERBATIM]** "This policy establishes standards to ensure that state agencies: •
> Appropriately analyze and document the benefits, costs, and risks to the state before
> contracting for a Cloud or Hosted Service. • Assess the readiness of a Cloud or Hosted
> Service Provider to deliver a solution that meets the state's requirements. • Conduct
> planning to ensure that state information and financial assets are appropriately
> protected when adopting a Cloud or Hosted Service."

### Applicability

**[SUMMARY]** All state agencies as defined in ORS 276A.230 (Executive Branch boards,
commissions, departments, divisions, offices), with the exclusions listed in the source.
(Source: Applicability section.)
