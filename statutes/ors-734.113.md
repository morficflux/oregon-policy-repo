# Application of Oregon Receivership Code (ORS 734.113)

## At a glance

ORS 734.113 — Application of Oregon Receivership Code. Chapter 734 (Chapter 734), 2025 Edition.

## Full text

734.113 Application of Oregon Receivership Code. Notwithstanding ORS 37.040, the Oregon Receivership Code does not apply to delinquency proceedings under this chapter. [2017 c.358 §57] Note: 734.113 was added to and made a part of ORS chapter 734 by legislative action but was not added to any smaller series therein. See Preface to Oregon Revised Statutes for further explanation.
