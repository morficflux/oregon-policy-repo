## Full text

815.430 Submitting false odometer disclosure; penalty.

(1) A person commits the offense of submitting a false odometer disclosure if the person knowingly makes any false statement or provides any false information on an odometer disclosure form.

(2) The offense described in this section, submitting a false odometer disclosure, is a Class C felony. [1985 c.251 §7; 1991 c.873 §21] (Traffic Control Signal Operating Devices)
