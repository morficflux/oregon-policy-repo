# Declaration regarding state financial aid for students (ORS 350.615)

## At a glance

ORS 350.615 — Declaration regarding state financial aid for students. Chapter 350 (Chapter 350), 2025 Edition.

## Full text

350.615 Declaration regarding state financial aid for students. The Legislative Assembly declares that it is an important public interest, and an appropriate objective of the state, that Oregonians who need financial assistance to attend an independent institution in Oregon should have access to state financial aid. [Formerly 352.672]
