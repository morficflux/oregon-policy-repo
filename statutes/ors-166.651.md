## At a glance

## Full text

166.651 Throwing an object off an overpass in the first degree.

(1) A person commits the crime of throwing an object off an overpass in the first degree if the person: (a) Recklessly throws an object off an overpass; and (b) Knows, or reasonably should have known, that the object was of a type or size to cause damage to any person or vehicle that the object might hit.

(2) Throwing an object off an overpass in the first degree is a Class C felony. [1993 c.731 §2]
