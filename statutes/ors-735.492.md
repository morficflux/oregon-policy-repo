# Application of certain Insurance Code provisions to surplus lines insurers (ORS 735.492)

## At a glance

ORS 735.492 — Application of certain Insurance Code provisions to surplus lines insurers. Chapter 735 (Chapter 735), 2025 Edition.

## Full text

735.492 Application of certain Insurance Code provisions to surplus lines insurers. ORS 731.324, 731.328, 731.512 and 731.624 do not apply to surplus lines insurers. [2005 c.185 §17]
