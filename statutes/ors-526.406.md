# Adopting rules as part of rule package NONINDUSTRIAL PRIVATE LANDOWNER ASSISTANCE (Generally) (ORS 526.406)

## At a glance

ORS 526.406 — Adopting rules as part of rule package NONINDUSTRIAL PRIVATE LANDOWNER ASSISTANCE (Generally). Chapter 526 (Forestry Administration), 2025 Edition.

## Full text

526.406 Adopting rules as part of rule package. The State Board of Forestry shall adopt the rules described in ORS 526.400, 526.402 and 527.738 as part of the rule package described in ORS 527.711. [2022 c.33 §20] Note: 526.406 is repealed only if certain conditions are met. See sections 54 and 60 to 64, chapter 33, Oregon Laws 2022 (third note following 526.400).
