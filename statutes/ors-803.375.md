## Full text

803.375 False application prohibited; penalty.

(1) A person commits the offense of false application for vehicle registration if the person does any of the following: (a) Knowingly makes any false statement or representation with respect to any facts required to be set forth in any application for registration. (b) Uses a name other than the person’s true name in any application for registration.

(2) The penalty for submitting a false odometer reading in an application for registration is as provided in ORS 815.430.

(3) The offense described in this section, false application for vehicle registration, is a Class A misdemeanor. [1983 c.338 §212; 1985 c.16 §80; 1985 c.251 §19]
