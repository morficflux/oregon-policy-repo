# Prohibition against using marijuana item in public place (ORS 475C.377)

## At a glance

ORS 475C.377 — Prohibition against using marijuana item in public place. Chapter 475C (Chapter 475C), 2025 Edition.

## Full text

475C.377 Prohibition against using marijuana item in public place.

(1) It is unlawful for any person to engage in the use of marijuana items in a public place.

(2) A violation of subsection

(1) of this section is a Class B violation. [Formerly 475B.381]
