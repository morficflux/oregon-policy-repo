## Full text

807.580 Using invalid license; penalty.

(1) A person commits the offense of using an invalid license if the person knowingly displays or permits to be displayed or possesses any license or driver permit that the person knows is fictitious, canceled, revoked, suspended or fraudulently altered.

(2) The offense described in this section, using an invalid license, is a Class A misdemeanor. [1983 c.338 §340]
