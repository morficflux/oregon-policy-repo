# Failure to file brief by appellant; appearance by defendant (ORS 138.210)

## At a glance

ORS 138.210 — Failure to file brief by appellant; appearance by defendant. Chapter 138 (Chapter 138), 2025 Edition.

## Full text

138.210 Failure to file brief by appellant; appearance by defendant. If the appellant fails to file a brief in the appellate court, the court shall dismiss the appeal. The defendant need not personally appear in the appellate court.
