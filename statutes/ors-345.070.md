# Registered agent identification credential (ORS 345.070)

## At a glance

ORS 345.070 — Registered agent identification credential. Chapter 345 (Chapter 345), 2025 Edition.

## Full text

345.070 Registered agent identification credential. The employing school shall issue an identification credential to each agent registered under ORS 345.010 to 345.340 in the form and size prescribed by the Higher Education Coordinating Commission. Each agent shall carry the credential at all times while engaged as an agent of the school. [Amended by 1961 c.268 §8; 1975 c.478 §8; 1989 c.333 §7; 2012 c.104 §27]
